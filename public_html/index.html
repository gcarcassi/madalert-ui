Madalert: MaDDash mesh analysis

Report

Analyzes a single mesh and reports problematic patterns. Examples:

  • Latency tests between all WLCG hosts Dashboard (psmad.grid.iu.edu)
  • Latency tests between all WLCG hosts Dashboard (maddash.aglt2.org)
  • USATLAS Mesh Config

Diff

Analyzes two meshes and creates a new mesh comparing the two. Examples:

  • Latency tests between all WLCG hosts Dashboard (psmad.grid.iu.edu vs maddash.aglt2.org)
  • Dual-Stack Mesh Config (IPv4 vs IPv6)
